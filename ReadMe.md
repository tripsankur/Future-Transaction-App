# Read Me

ABN-AMRO-CodingChallenge.zip contains below list of files.
Please update the metadata location in the config file and the log file location in the logging file before running the app.


------------------------------------------------------------------------------------------------------------------
- **Future-Transaction-Writer:** This is the complete Maven project which can be imported in eclipse.

- **Future-Transaction-Writer-1.0.jar :** Zip fils contain the actual build jar which can be used to execute the application.

- **metadata :** Metadata file. Path of this file must be included/updated in the config.

- **config.properties :** configuration which needs to be provided to the app while running. 

- **log4j.properties :** log4j property. Log file path can be updated while running the app. 

- **logging.log :** Log file generated by the app.

- **Output.csv :** Output file generated by the app.

- **App_Details.md :** Document containg the details about the Application.

------------------------------------------------------------------------------------------------------------------
